# HR Analytics Dashboard (Excel)

## Project Overview
This project demonstrates an interactive HR Analytics Dashboard created using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Features
- Employee Overview
- Department-wise Analysis
- Gender Distribution
- Attrition Analysis
- KPI Dashboard
- Interactive Filters
- Business Insights

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Dashboard Design
- KPI Reporting
- Data Visualization
- Analytical Thinking

## Outcome
Created an interactive dashboard that helps HR teams analyze employee data and support data-driven decision-making.
